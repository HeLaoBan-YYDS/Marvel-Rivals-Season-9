## 2、网站主题色建议

结论：**默认选择暗色主题**。Marvel Rivals Season 9 适合用深色背景、红色主色、青蓝/金色点缀，贴合英雄射击、漫画、Thebes 古埃及神秘感和高对比电竞 UI。

```css
/* 导航页主题色 - 亮色主题 */
--nav-theme: 358 80% 52%;        /* Marvel crimson red, 接近 #E62429 */
--nav-theme-light: 358 88% 62%;  /* 更亮的红色，用于 hover / active */

/* 导航页主题色 - 暗色主题 */
--nav-theme: 358 80% 52%;        /* crimson red */
--nav-theme-light: 190 95% 58%;  /* electric cyan，用于暗色背景上的能量高光 */
```

辅助色建议：

```css
--page-bg: 225 28% 8%;           /* 深海军蓝黑 */
--card-bg: 225 22% 12%;          /* 深色卡片 */
--accent-gold: 42 90% 58%;       /* Thebes / Egyptian gold */
--accent-cyan: 190 95% 58%;      /* Regenerative shield / energy blue */
--text-main: 0 0% 96%;
--text-muted: 220 14% 72%;
```
